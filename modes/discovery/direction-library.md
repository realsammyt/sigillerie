# direction-library

## Orientation

This file is the canonical taste backbone for Sigillerie. Discovery Phase 3 reads it to pick three differentiated visual directions, one per school, rendered as one-page HTML lookbook tiles. Producer mode reads the chosen entry as part of `design-context.md`. Data viz and knowledge-graph capabilities read it for style guidance. Twenty philosophies, grouped into five schools. Each entry is a real designer or studio with a real lineage. No invented movements.

---

## The Five Schools

Each school is a row in the 5×4 grid. Phase 3 picks across rows, never within. If vibe-words rule out a row, substitute from the closest neighbor school.

| # | School | Core stance | Best surfaces |
|---|---|---|---|
| 1 | Information Architecture | Data is structure, type is voice, grid is law | Editorial sites, dashboards, identity systems, decks |
| 2 | Motion Poetics | Code and motion are the medium, not decoration | Hero sites, launch pages, AR/3D, kinetic posters |
| 3 | Eastern Minimalism | Whitespace carries meaning, restraint is the loudest move | Premium product, lifestyle brands, ritual UI |
| 4 | Experimental Avant-Garde | Break the rule to find the form, the hand stays visible | Cultural brands, music, fashion, art books |
| 5 | Magazine / Editorial | Reading is the experience, hierarchy is dramaturgy | Long-form, lifestyle, hospitality, content brands |

**School 1 anchors**: Pentagram (Michael Bierut, Paula Scher, Natasha Jen), Massimo Vignelli, Information Architects (Oliver Reichenstein), Ruder/Müller-Brockmann lineage, Apple typography era 1989–2000 (Susan Kare, Bruce Tognazzini contemporaries).

**School 2 anchors**: Field.io (Marcus Wendt, Vera-Maria Glahn), Active Theory (Michael Anderson), Resn, Locomotive, Hort, Build in London for kinetic identity, Zach Lieberman for code-poetics.

**School 3 anchors**: Kenya Hara (MUJI art direction), Takram (Kotaro Watanabe), Kashiwa Sato (Uniqlo, Tsutaya), Issey Miyake studio, Naoto Fukasawa, Nendo (Oki Sato), Yugo Nakamura.

**School 4 anchors**: Stefan Sagmeister, David Carson (Ray Gun era), Wolfgang Weingart (Basel), Cranbrook Academy 1980s under Katherine McCoy, Irma Boom, April Greiman, Neville Brody (The Face, Fuse).

**School 5 anchors**: Aperture, Apartamento (Omar Sosa, Marco Velardi), Tyler Brûlé / Monocle, T: The New York Times Style Magazine (Patrick Li, Ina Saltz lineage), Pentagram editorial (Luke Hayman, Jen at Wired), Fabien Baron at Interview, Kinfolk.

## Differentiation Rules for the 3-Direction Shortlist

Phase 3 must return three directions that look genuinely different. Without rules the agent regresses to three minimalisms. The rules:

**Rule 1, Cross-school requirement.** The three directions come from three different schools out of five. Never two from the same row. A shortlist of {Pentagram-Typographic IA, iA Content-First, Vignelli-Modernist Grid} fails the rule, all three are School 1.

**Rule 2, Vibe-word filter first.** Before picking, score each school against the user's three vibe-words from intake. If "calm" or "quiet" lands, School 3 is in. If "bold" or "loud" lands, Schools 4 and 5 contend. If "technical" or "data-heavy" lands, School 1 leads. If "alive" or "kinetic" lands, School 2 leads.

**Rule 3, Anti-brand exclusion.** Whatever the user named in intake question 5 ("what you're not") rules out the closest matching school. If they said "not corporate", strike School 1's Vignelli entry. If "not edgy", strike School 4. The strike happens first, before scoring.

**Rule 4, One safe, one stretch, one curveball.** Of the three, one direction should be the obvious read of the brief, one should pull the brief one step further, one should be a controlled risk from a school the user did not anticipate. Phase 3 labels these in the lookbook tiles as `safe`, `stretch`, `curveball` so the user knows what they are choosing between.

**Rule 5, Reference-substitution check.** If the user pasted reference URLs in Phase 2 that already match one of the three picks, swap that pick for a different philosophy in the same school. Discovery's job is to show options, not mirror back what they already brought.

**Rule 6, Surface-fit veto.** If the first delivery surface (intake question 6) is App Store screenshots, School 4 (Carson-Deconstructed especially) is vetoed. If the surface is AR or 3D, School 1's strict-grid entries deprioritize. Surface trumps vibe when they conflict.

**Default cross-school template.** When in doubt, the template that works most often: one School 1 entry (structure), one School 2 or 3 entry (atmosphere), one School 4 or 5 entry (voice). That triangulation hits structure, mood, and personality.

## Anti-Patterns

Avoid these shortlist constructions. They look like differentiation but are not.

**1. Three minimalisms.** Shortlists like {Hara-Whitespace, iA Content-First, Nendo-Object-Quiet} feel like three distinct picks but they all collapse to "white page, small type". The user gets no real choice. Fix: keep one minimalism, pair it with a maximalist (Sagmeister, Carson, Brody) and a structural (Pentagram, Vignelli).

**2. "Avant-garde" as a generic direction.** Saying "let's do something experimental" with no concrete designer attached. Phase 3 must always cite a real philosophy entry. If School 4 is in the shortlist, it is specifically Sagmeister-Visceral OR Carson-Deconstructed OR Weingart-Basel-New-Wave OR Brody-Face-Era. Pick one. The differences matter.

**3. Mirror of the user's reference.** The user pasted Linear's site as a reference. The shortlist comes back with three Pentagram-flavored picks. The user already had that direction before Discovery started. Phase 3 must show options the user did not bring.

**4. Three picks from the agent's training-data favorites.** If the agent always returns Pentagram + Hara + Sagmeister regardless of brief, that is bias not curation. Rotate. The 20 entries are equally weighted.

**5. Naming a studio that is not in this file.** If the agent suggests "Pentagram-style" without citing one of the 20 named entries, the suggestion is too loose. The lookbook tile must point at one row of the table.

**6. Using "minimalist" or "modern" as the rationale.** Both words are noise. The rationale field in the tile must reference the philosophy's specific signatures, not generic adjectives.
